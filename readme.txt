== Description ==

Social Flyer is a responsive fancy social follow WordPress plugin. Social Flyer Fly in to the center of the document(Website) on page scroll which grabs attention from website visitor. Social Flyer Do not require any social API Key as it is just URL link to Profile/Any page.

You just have to give your social profile URL. It is not mandatory to use only social URL, you can place any URL, for example: URL to your tutorial page etc. Since it is a URL Redirect, you can convert your traffics/visitors to maximum of 8 URL (Design holds maximum of 8 Elements).

== Installation ==

1. Download the Social Flyer Plugin 
2. Go to Plugins -> Add New
3. Click Upload Plugin
4. Browse socialflyer.zip Zip file and then upload it by clicking Install Now
5. Click Activate Plugin thats it.

== Frequently Asked Questions ==

= Does Social Flyer needs any API credentials =

Absolutely Not. It does not need any API details

= Does Social Flyer work on social plugins =

No it does not included any social plugin. Its functionality is just redirection to the specified URL

= How statistics are calculated ? =

Since it is just a redirection the statistics are calculated based on user click to the specific channel, that's why the naming are done as social network 1,2, etc to make sure the statistics are not confusable.
